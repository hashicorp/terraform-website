## Automate Terraform

Name
Description
Managed By
 [`terraform-exec`](https://github.com/hashicorp/terraform-exec)
Use the Go module to construct and run CLI commands. Refer to the `terraform-json` repository (below) for documentation about the return values.
HashiCorp
 [`terraform-json`](https://github.com/hashicorp/terraform-json)
Parse the output from CLI commands `terraform show -json`(/docs/commands/show.html#json-output) and [`terraform providers schema -json`](/docs/commands/providers/schema.html#json) and receive documentation for the output format. You can also use the provided helpers to work with the data.
HashiCorp
[Terraform Orb](https://circleci.com/developer/orbs/orb/circleci/terraform)
Deploy your Terraform infrastructure via a CircleCI pipeline. CircleCI Orbs let you package, ship, and re-use configurations.
**Hands On:** Get started with the [Deploy Terraform infrastructure with CircleCI](https://learn.hashicorp.com/tutorials/terraform/circle-ci) tutorial on HashiCorp Learn.
Third Party
[GitHub Actions for Terraform](https://github.com/hashicorp/setup-terraform)
Automate Terraform in your GitHub Actions workflow. After you have used the action, subsequent steps in the same job can run Terraform commands using the GitHub Actions run syntax. This allows most Terraform commands to work exactly like they do on your local command line.


**Hands On:** Try the [Automate Terraform with GitHub Actions](https://learn.hashicorp.com/tutorials/terraform/github-actions) tutorial on HashiCorp Learn.
Third Party

## HashiCorp Products
You can use HashiCorp products together to enable self-service automation across teams and organizations. Watch CEO and Co-founder Armon Dadgar discuss how you can use Packer, Terraform, and Consul together to enable modern infrastructure automation.

Description
